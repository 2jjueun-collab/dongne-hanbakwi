# 동네 한 바퀴 · 하곰 탐험 MVP

GitHub Pages 또는 Vercel에서 바로 배포할 수 있는 정적 웹앱입니다.

## 포함 기능

- 경산 랜드마크 GPS·QR·데모 방문 인증
- 같은 장소는 하루 1회만 인증 가능하며 날짜가 바뀌면 재방문 가능
- 장소별 랜덤 아이템 지급 및 누적 방문 횟수 기록
- 투명 배경의 정면 하곰 캐릭터와 아이템 꾸미기
- 포인트 상점과 비공개 히든 아이템 뽑기
- 공유 코드 기반 친구 추가와 친구 하곰 확인
- 친구 3명 달성 시 숨겨진 우정 뱃지
- 휴대전화 모션 센서를 이용한 일일 만보기
- 1,000걸음마다 10P 자동 지급
- 걸음 수, 추정 거리, 추정 칼로리, 10,000걸음 목표 표시
- 친구 공유 코드에 오늘 걸음 기록 포함
- 나와 친구의 오늘 걸음 순위 표시
- 발표용 +1,000걸음 및 친구 3명 추가 버튼

## Vercel 배포

1. 폴더의 모든 파일을 GitHub 저장소에 업로드합니다.
2. Vercel에서 해당 저장소를 Import합니다.
3. Framework Preset은 `Other`, Build Command는 비워 두고 Output Directory는 `.`로 설정합니다.
4. 배포 후 이전 화면이 남으면 `Ctrl + Shift + R`로 강력 새로고침합니다.

## 만보기 사용 조건

- HTTPS로 배포된 휴대전화 브라우저에서 사용합니다.
- iPhone Safari는 사용자가 버튼을 누른 뒤 모션 센서 권한을 허용해야 합니다.
- 웹 만보기는 브라우저가 열려 있는 동안 동작하는 MVP입니다. 화면이 꺼지거나 브라우저가 백그라운드로 이동하면 운영체제에 따라 측정이 중단될 수 있습니다.
- 거리와 칼로리는 각각 1걸음당 0.68m, 0.04kcal를 적용한 단순 추정치입니다.

## 친구 걸음 순위 방식

이 버전은 데이터베이스가 없는 정적 MVP이므로 친구의 기록이 실시간으로 자동 갱신되지는 않습니다. `친구에게 공유` 또는 `내 하곰 공유 코드`에서 생성한 새 코드를 친구가 다시 추가하면, 오늘 걸음 수와 착용 아이템이 갱신됩니다.

실시간 동기화가 필요하면 Supabase 또는 Firebase를 연결해야 합니다.


## 아이템 시각 시스템
- 아이템은 운영체제별 이모지 대신 `assets/items/*.svg` 벡터 아이콘을 사용합니다.
- 착용 위치와 크기는 `data.js`의 각 아이템 `placement` 값으로 개별 조정할 수 있습니다.
- `x`, `y`, `width`는 하곰 캐릭터 영역 대비 백분율이며 `rotate`는 회전 각도입니다.


## v11 안정화 수정

- 아이템 SVG를 `data.js`에 내장해 `assets/items` 폴더가 GitHub 업로드 과정에서 누락되어도 아이콘이 깨지지 않습니다.
- 하단 메뉴의 고정 레이아웃 스타일을 복원했습니다.
- 서비스 워커는 HTML/CSS/JS에 네트워크 우선 방식을 사용해 새 배포가 이전 캐시에 가려지는 문제를 줄였습니다.


## v12 모바일 화면 수정
- 360~620px 화면에서 홈 히어로 영역을 1열로 전환
- 친구 코드, 입력창, 공유 코드의 가로 넘침 방지
- 하곰 꾸미기, 상점, 만보기, 순위, 도감의 모바일 레이아웃 보정
- 하단 7개 메뉴를 화면 너비 안에 고정
- 누락된 모달 및 상점 공통 스타일 복구

## v13 모바일 친구 카드 수정

- 친구 하곰 미리보기가 텍스트 영역을 침범하던 문제 수정
- 모바일에서 친구 카드 이미지 영역을 고정하고 캐릭터를 내부에 맞춤
- 친구 이름, 걸음 수, 아이템 목록의 줄바꿈과 말줄임 처리
- 하단 고정 메뉴에 마지막 친구 카드가 가려지지 않도록 여백 확대
- 서비스 워커 캐시 이름과 CSS 버전 쿼리 갱신


## v14 장소별 일일 인증 제한

- 같은 랜드마크는 현지 날짜 기준 하루에 한 번만 GPS·QR·데모 인증 가능
- 하루 동안 여러 서로 다른 장소를 인증하는 것은 가능
- 인증 완료 장소는 `오늘 완료`로 표시되고 버튼이 비활성화됨
- 날짜가 바뀌면 같은 장소를 다시 방문해 랜덤 아이템과 포인트 획득 가능
- 누적 방문 횟수와 기존 아이템·포인트·친구·걸음 데이터 유지
- 앱을 자정 이후 다시 열거나 포커스하면 일일 상태 자동 갱신
